# Vue.js高仿饿了么外卖App课程总结


历时一个多月终于学完了黄轶老师的这门Vue课程，期间考试什么的耽误了不少，总的来说肯定是受益匪浅，感受到了Vue实战的魅力。以前总是看文档，敲小demo，不仅感受不到文档的重点，怎么去运用，还容易学了忘，忘了学。借着昨天刚刚学完的契机，趁热打铁，写一点自己的收获，做个总结，方便以后的查阅。
